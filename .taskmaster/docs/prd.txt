## 项目：Hyperliquid 相关系数异常监控与告警

### 目标
创建一个专注于分析 Hyperliquid 交易所中山寨币与 BTC/USDC:USDC 的皮尔逊相关性的系统，识别"短期低相关但长期高相关"且可能存在时间差（滞后关系）的异常币种，通过飞书机器人实时推送告警。

### 用户故事
- 作为交易者，我希望能够自动识别与 BTC 存在滞后关系的山寨币，以便发现潜在的套利或补涨机会。
- 作为开发者，我希望系统能够高效地处理大量交易对的数据分析，减少 API 调用次数。
- 作为运维人员，我希望系统能够稳定运行，并在发现异常时及时推送告警。

### 功能
1. **数据获取**：通过 REST API 获取 Hyperliquid 交易所的历史 K 线数据
2. **数据缓存**：使用 SQLite 持久化数据，支持增量更新
3. **相关性分析**：计算不同时间周期（1d, 7d, 30d, 60d）的皮尔逊相关系数
4. **延迟检测**：计算最优延迟 τ*，识别滞后关系
5. **异常识别**：识别短期低相关但长期高相关的异常模式
6. **告警推送**：通过飞书机器人推送异常告警
7. **持续监控**：支持定时任务，持续监控市场变化

### 技术栈
- Python 3.12+
- ccxt: 加密货币交易所 API
- pandas/numpy: 数据分析
- SQLite: 数据持久化
- Redis: 告警去重（可选）
- WebSocket: 实时数据订阅（扩展功能）
